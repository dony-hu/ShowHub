# ShowHub - 丰图黑板报与产品站

一个现代化的 Web 应用，用于展示丰图科技的产品进展、技术文章与行业洞察，承载公司技术文化的「黑板报」。

## 项目目标

1. **对齐业界标准**：参考高德、百度、Google 等主流图商，推进 API 命名、路径、认证与错误反馈的标准化治理。
2. **透明反馈机制**：通过「地图纠错」与「产品缺陷」模块，建立从用户上报到研发修复的闭环流程。
3. **开发者深度共建**：依托「功能改进」建议池，由社区开发者投票决定产品演进优先级，打造更友好的 LBS 开发生态。
4. **极简集成体验**：持续优化文档质量，提供扁平、语义化、RESTful 的接口设计，降低多图商平台接入成本。

## 功能特性

### 📰 黑板报（技术文化）
- 内部快讯：团队分享、圆桌与实战笔记（周更）
- 技术趋势：研发一线的可落地实验与方向
- 产品故事：版本迭代抉择、架构思路与复盘
- 行业展望：值得下注的赛道与策略
- 文章详情：支持 Markdown 渲染与沉浸阅读
- 登录发帖：内部成员登录后可发布文章
- 标签与检索：按主题快速浏览

### 🎨 设计特点
- 现代化的UI设计，采用渐变色
- 响应式布局，支持移动设备
- 流畅的过渡动画
- 深色导航栏，浅色内容区

## 技术栈

- **React 18** - UI框架
- **TypeScript** - 类型安全
- **React Router** - 页面路由
- **Vite** - 构建工具
- **CSS3** - 样式和动画

## 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式运行
```bash
npm run dev
```

访问 `http://localhost:5173` 查看应用

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```

## 使用说明

### 打开黑板报
1. 启动开发服务后，访问导航栏中的「黑板报」
2. 浏览各分区内容（内部快讯 / 技术趋势 / 产品故事 / 行业展望）
3. 登录后点击「我要分享」发布新文章
